# Book Recommender

A simple web-based Book Recommendation System that helps users discover books based on mood, genre, and search. It also allows users to save their favorite books.

---

## Features

* Search books by title or author
* Filter books by genre and mood
* Add and remove favorites (stored using localStorage)
* Dark mode toggle
* Expandable book descriptions (Read More / Read Less)
* Redirect to Amazon with confirmation

## How It Works

* Book data is stored in an array of objects
* JavaScript `filter()` is used for search and recommendations
* `forEach()` is used to display books dynamically
* Favorites are stored using browser localStorage
